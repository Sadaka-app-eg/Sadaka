## ✨ عن التطبيق

**"كن ذا أثر"** مش مجرد تطبيق أذكار عادي... ده منظومة إيمانية متكاملة بُنيت بحب لتكون رفيقك اليومي في رحلة الاستقامة. من قراءة القرآن بمصحف تفاعلي، لمجتمع اجتماعي إسلامي كامل بالصداقات والمحادثات، لشعلة حية بتعكس نشاطك الإيماني، لشجرة نعم بتكبر معاك يوم بعد يوم — كل التفاصيل اتصممت عشان تخلي طاعتك أسهل وأجمل وأكتر استمرارية.

> 💡 التطبيق شغال بالكامل بتقنية **PWA** يعني بيتثبت على موبايلك زي أي تطبيق عادي من غير ما يحتاج متجر تطبيقات.

## 🚀 المميزات الرئيسية

### 📖 القرآن الكريم — مصحف تفاعلي بمستويين
- **وضعان للقراءة:** نص عثماني نظيف قابل للتحديد، أو **مصحف مصور (QCF)** بشكل الصفحات الحقيقي
- **10 قراء مختلفين** للاستماع: المنشاوي، الحصري، العفاسي، عبد الباسط، ماهر المعيقلي، وغيرهم
- **تحديد ذكي للآيات** بالضغط المطول + تشغيل نطاق كامل مع خيار **التكرار المخصص**
- **بحث فوري** في السور والآيات معاً بالكلمة
- **الضغطة المطولة على السورة** بتفتحلك أسباب النزول وتدبر السلف الصالح 📜
- **علامة "وقفت هنا"** 🔖 لحفظ آخر نقطة قراءة والرجوع لها من الصفحة الرئيسية
- **وضع القراءة الليلية** للمصحف بعكس الألوان لراحة العين
- **زر ملء الشاشة** لتجربة قراءة بلا تشتيت
- شريط تقدم صوتي بالسحب المباشر (Seekbar) لتقديم وتأخير التلاوة

### 🤲 الأذكار الكاملة
- أذكار الصباح، المساء، النوم، الاستيقاظ، وبعد الصلاة — بالحديث والمصدر الموثق لكل ذكر
- عداد تفاعلي لكل ذكر مع **شريط تقدم مرئي** وشارة "✓ تم" عند الاكتمال
- تشغيل صوتي مدمج لأذكار الصباح والمساء بصوت الشيخ العفاسي

### 📿 السبحة الإلكترونية
- 9 صيغ تسبيح جاهزة + عداد حي بحركة "نبض" عند كل ضغطة
- شريط تقدم دائري لكل 33 تسبيحة، وتتبّع للإجمالي الكلي

### 🔥 نظام "شعلة الأثر" — رفيقك الحي
واحدة من أجمل اللمسات: شعلة افتراضية بتنمو أو بتخفت حسب نشاطك الإيماني اليومي!
- من `💨` (خافتة) → `🕯️` → `🔥` → `✨` (متوهجة كاملة)
- شريط حيوية (Vitality) من 0 إلى 100 بيتحرك بسلاسة
- كل تسبيحة، ذكر، أو نعمة مكتوبة بتغذي الشعلة وتخليها تتوهج أكتر

### ❤️ دفتر النعم — شجرة تكبر بأثرك
- اكتب نعمة واحدة كل يوم، وشوف **شجرة حقيقية بتنبت أوراق** 🍃 مع كل نعمة تكتبها
- عند الوصول لمحطات معينة (10، 50، 100، 365 نعمة) — الشجرة **تُثمر** 🍎 باحتفال خاص
- كل ورقة قابلة للضغط لعرض النعمة المكتوبة وتاريخها

### 🌑 الأثر الخفي — صدقة السر
مستوحى من حديث "رجل تصدق بصدقة فأخفاها حتى لا تعلم شماله ما تصدقت به يمينه":
- سجّل أعمالك الخيرية سراً، واختر **متى تُكشف** — إما سنوياً أو بعد عدد أعمال معين
- شاشة "حجاب" مقفولة بأنيميشن مؤثر لحين موعد الكشف
- لحظة الكشف بيتحول لتجربة بصرية مؤثرة بستارة (Veil Curtain) متحركة

### 👥 مجتمع أثر — شبكة اجتماعية إسلامية كاملة
منظومة متكاملة مبنية على **Firebase Firestore** بتحديثات لحظية (Real-time):
- 📝 **ساحة الأثر:** نشر فوائد بالنص والصور مع تفاعلات (❤️👍🤝😮😢😡) وتعليقات حية
- 💬 **مجلس الذكر:** شات جماعي مباشر منفصل للرجال والنساء + رسائل صوتية
- 🤝 **نظام صداقة كامل:** إرسال/قبول/رفض طلبات صداقة، وقائمة أصدقاء
- 🟢 **حالة "متصل الآن"** حقيقية بنظام Heartbeat كل دقيقة
- 📥 **رسائل خاصة** ثنائية بستايل واتساب، بالصور والصوت
- 🕌 **حملة استيقاظ الفجر:** سجّل رقمك ليوقظك متطوع، أو كن أنت المتطوع
- 🏡 **التنافس العائلي:** أنشئ غرفة عائلية بكود دعوة، وتتبع الورد والأذكار مع تحديات مخصصة تبتكرها بنفسك
- 🌟 **حلقة الأسبوع:** تحدي جماعي أسبوعي يتغير تلقائياً كل أسبوع
- 🏆 **لوحة الشرف الشهرية:** ترتيب حي لأكثر الأعضاء تفاعلاً، تتصفر تلقائياً كل شهر
- 🤲 **اطلب دعاء:** بخيار النشر باسمك أو بشكل مجهول
- ✨ **الأكثر تأثيراً:** أرشيف تلقائي للمنشورات اللي وصلت لحد معين من التفاعل
- 🎖️ **نظام نقاط ورتب:** "مبادر الطيب 🌿" → "ذاكر مجتهد 🌟" → "منارة الأثر 🏆"
- 🔥 **نظام Streak يومي** لتتبع الأيام المتتالية من النشاط

### 🕌 مواقيت الصلاة والقبلة
- حساب المواقيت تلقائياً حسب **الموقع الجغرافي الفعلي**
- **عداد تنازلي حي** بالثواني للصلاة القادمة داخل دائرة متحركة
- 🧭 **بوصلة اتجاه القبلة** حقيقية بالحساس (Gyroscope) مع سهم يتبع حركة الجهاز
- 📍 زر "أقرب مسجد" يفتح خرائط جوجل فوراً بموقعك
- 📢 **نظام أذان صوتي تلقائي** مع اختيار مؤذن مختلف للفجر عن باقي الصلوات

### 📿 الختمة المباركة + خطة الحفظ
- اختر مدة ختمتك (شهر / شهرين / 3 شهور / مخصص) وتابع تقدمك سورة بسورة
- **خطة حفظ ومراجعة ذكية** تحسب لك ورد كل يوم بناءً على أيام إجازتك الأسبوعية

### 📚 المكتبة العلمية — موسوعة بـ16 قسم
الفقه الميسر، أحاديث صحيحة، قصص الأنبياء، سير الصحابة، العقيدة، السيرة النبوية، رقائق وإيمانيات، سير العلماء، اختبارات قصيرة تفاعلية، وأكتر!

### 🌿 أبواب العلم + أكاديمية أثر
موسوعة شرعية ميسّرة بالدليل، ومسار تعليمي ممنهج على مستويات متدرجة لبناء أساس علمي راسخ

### 💭 حالتي اليوم — دعم نفسي إيماني
اختر حالتك المزاجية (فرحان، مبسوط، عادي، زعلان، مكتئب) وهيجيلك:
- آيات وأحاديث تناسب حالتك تحديداً
- خواطر وتأملات مكتوبة بعناية
- نصائح عملية قابلة للتطبيق فوراً
- دعاء مخصص لحالتك

## 🖌️ التفاصيل التقنية والحركات (Animations)

التطبيق مليان بلمسات حركية بسيطة بس بتفرق:

```css
@keyframes pulse       /* نبضة النقطة الخضراء "متصل الآن" */
@keyframes fadeIn      /* ظهور ناعم للمودالز والكروت */
@keyframes float       /* تحليق الهلال في كارت الهوية */
@keyframes leafGrow    /* نمو أوراق شجرة النعم */
@keyframes fruitGrow   /* ثمار المحطات الكبرى في الشجرة */
@keyframes flameFlicker/* ارتعاش شعلة الأثر الحية */
@keyframes lockPulse   /* نبض قفل الأثر الخفي */
@keyframes islamicPulse/* توهج الهلال في شاشة البداية */
```

- **منيو جانبي (Drawer)** ينزلق بسلاسة من اليمين بـ `cubic-bezier` احترافي
- **Action Sheets** تنزلق من الأسفل زي تطبيقات iOS الأصلية
- **Long Press Detection** ذكي يفرّق بين السكرول والضغطة المطولة بدقة بكسل
- بنية الصفحات كاملة **Single Page App** بدون أي إعادة تحميل، مع إدارة `history.pushState` للتحكم في زر الرجوع بشكل طبيعي

## 📲 التثبيت والتشغيل

1. حمّل المشروع أو اعمل `clone` للريبو
2. افتح ملف `index.html` مباشرة، أو استضفه على أي خادم استضافة ثابت (Static Hosting)
3. أضف بيانات Firebase الخاصة بيك في `firebaseConfig`
4. من على الموبايل: افتح الرابط في المتصفح → **"إضافة إلى الشاشة الرئيسية"** وهيتثبت زي أي تطبيق عادي 📱

```bash
git clone https://github.com/your-username/kon-za-athar.git
cd kon-za-athar
# افتح index.html في المتصفح أو استخدم أي سيرفر محلي
```
